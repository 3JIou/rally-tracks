## Geopoint track

This track is based on [PlanetOSM](http://wiki.openstreetmap.org/wiki/Planet.osm) data. 

### Example Document

```json
{
  "location": [
    -0.1485188,
    51.5250666
  ]
}
```

### Parameters

This track allows to overwrite the following parameters with Rally 0.8.0+ using `--track-params`:

* `bulk_size`
